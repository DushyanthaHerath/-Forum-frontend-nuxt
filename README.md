## How to use

- Clone: __git clone https://github.com/DushyanthaHerath/-Forum-frontend-nuxt.git frontend__
- cd frontend
- Run __npm install__
- Run __npm run dev__

Frontend application is configured to the backend by port 4500. You can change the port from the nuxt.config file
